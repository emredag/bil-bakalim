## Why Archived?

These documents served their purpose during the project's development phase (2024-2025). They contain valuable historical context and design decisions, but the information has been:

1. **Implemented** - Features and designs have been built into the application
2. **Consolidated** - Key information moved to active documentation (USER_GUIDE.md, DEVELOPER_GUIDE.md)
3. **Superseded** - Current codebase is the source of truth
