# Task 18: Pause System

## Description
Implement game pause functionality with overlay and controls.

## Requirements from PRD
- **Section:** 4.6 Oyun Mekanikleri → Pause (Duraklat)

## Pause Behavior
- Tüm oyun dondurulur
- Blurred overlay + "Duraklatıldı" mesajı
- [Devam Et] [Ana Menü] butonları
- Süre sayacı durur

## Features
- Pause/resume toggle
- Game state frozen
- Timer stopped
- Blurred background overlay
- Resume and quit options

## Acceptance Criteria
- [ ] Pause button works (⏸ or Space key)
- [ ] Game completely frozen when paused
- [ ] Timer stops during pause
- [ ] Blurred overlay shown
- [ ] "Duraklatıldı" message displayed
- [ ] Resume button continues game
- [ ] Main menu button exits with confirmation
- [ ] Keyboard shortcut (Space) works
