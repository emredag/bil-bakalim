# Task 42: Application Icons

## Description
Create and configure application icons for all platforms.

## Requirements from PRD
- **Section:** 17.3 Uygulama İkonları

## Required Icon Sizes
- 32x32, 128x128, 256x256, 512x512 (PNG)
- ICO (Windows)
- ICNS (macOS)
- PNG (Linux)

## Icon Design
- Ana renk: Mavi/Mor gradient
- Kelime/harf teması
- Modern, minimal stil
- Tüm platformlarda tanınabilir

## Acceptance Criteria
- [ ] Icon designed (blue/purple gradient, word/letter theme)
- [ ] PNG icons: 32x32, 128x128, 256x256, 512x512
- [ ] ICO file for Windows
- [ ] ICNS file for macOS
- [ ] PNG for Linux
- [ ] Icons placed in correct Tauri directories
- [ ] Icons display correctly on all platforms
- [ ] Icon recognizable at all sizes
