# Task 22: Results Screen - Team Mode

> See also: [UI/UX Design Spec](../ui-ux-design.md#results)

## Description

Implement results screen for team mode with team rankings and player details.

## Requirements from PRD

- **Section:** 4.7 Sonuç Ekranı → Takım Yarışması

## Components

- Kazanan takım vurgusu (büyük)
- Takım sıralaması
- Her takım için toplam puan
- Detaylı gösterim
- Aksiyon butonları aynı

## Team Features

- Winning team highlighted
- Team rankings with total scores
- Team member contributions visible
- Team colors/emojis displayed

## Acceptance Criteria

- [ ] Winning team prominently displayed
- [ ] Team rankings shown
- [ ] Team total scores calculated
- [ ] Team member details expandable
- [ ] Team colors/emojis shown
- [ ] Action buttons work
- [ ] Results auto-saved with team data
